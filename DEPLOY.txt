====================================================
   دليل نشر الموقع مجاناً — خطوة بخطوة
====================================================

المواقع المستخدمة (كلها مجانية):
  - GitHub    : رفع الكود
  - Turso     : قاعدة البيانات السحابية
  - Vercel    : استضافة الموقع

====================================================
الخطوة 1 — إنشاء حساب GitHub
====================================================
1. روح على https://github.com
2. اضغط "Sign up" وأنشئ حساب مجاني
3. بعد الدخول اضغط "New repository"
4. اسم المستودع: kids-world
5. اتركه Public واضغط "Create repository"
6. اتبع التعليمات لرفع المشروع

====================================================
الخطوة 2 — إنشاء قاعدة بيانات Turso (مجانية)
====================================================
1. روح على https://turso.tech
2. اضغط "Get started free" وسجّل بحساب GitHub
3. بعد الدخول اضغط "Create database"
4. اختر أي اسم مثل: kids-world-db
5. اختر المنطقة: eu-central (أقرب للجزائر)
6. اضغط "Create database"

بعد الإنشاء:
7. اضغط على قاعدة البيانات
8. اضغط "Generate token" — انسخ هذا الـ Token واحفظه
9. انسخ أيضاً رابط قاعدة البيانات (يبدأ بـ libsql://)

====================================================
الخطوة 3 — نشر على Vercel (مجاني)
====================================================
1. روح على https://vercel.com
2. اضغط "Sign up" وادخل بحساب GitHub
3. اضغط "Add New Project"
4. اختر مستودع kids-world من GitHub
5. قبل الضغط Deploy، اضغط على "Environment Variables"
6. أضف هذه المتغيرات:

   الاسم                 | القيمة
   ---------------------|---------------------------
   ADMIN_PASSWORD       | كلمة السر التي تريدها
   DATABASE_URL         | libsql://اسم.turso.io
   TURSO_AUTH_TOKEN     | الـ Token من Turso
   NEXTAUTH_SECRET      | أي نص عشوائي طويل

7. اضغط "Deploy" وانتظر دقيقتين
8. Vercel يعطيك رابط مثل: https://kids-world.vercel.app

====================================================
الخطوة 4 — إضافة المنتجات لقاعدة البيانات السحابية
====================================================
بعد النشر، افتح لوحة التحكم:
  https://kids-world.vercel.app/admin

وأضف المنتجات يدوياً من لوحة التحكم.

====================================================
ملاحظات مهمة
====================================================
- Vercel المجاني يسمح بـ 100GB bandwidth شهرياً
- Turso المجاني يسمح بـ 500MB و1 مليار قراءة شهرياً
- كلاهما مجاني تماماً للمشاريع الصغيرة والمتوسطة
- يمكنك لاحقاً ربط دومين خاص بك مثل: kidworld.dz
